# Tides of Manila: A Merchant's Quest

![C Current](https://img.shields.io/badge/C-%3E%3D99-blue?style=flat-square)

Tides of Manila: A Merchant's Quest is a turn-based CLI-based game where you trade with various ethnic groups to become the most successful merchant in Manila.

## Usage

> This game needs to be compiled from the source files first, use [CMake](https://cmake.org/) to do this.
>
> ```bash
> cmake -B ./build -S .
> ```
>
> Instructions on how to use this program will be provided along the way.

```bash
./bin/ccprog1
```
